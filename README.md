# eruto-skills marketplace

Claude Code plugin marketplace for the [eruto-skills](https://github.com/eruto-skills) organization.

## Installation

```
/plugin marketplace add eruto-skills/marketplace
/plugin install research-note@eruto-skills
/plugin install marp@eruto-skills
/plugin install nanj-thread@eruto-skills
/plugin install design-deai@eruto-skills
/plugin install design-brief@eruto-skills
/plugin install ui-visual-reviewer@eruto-skills
```

See the [Claude Code plugin docs](https://docs.claude.com/en/docs/claude-code/plugins) for details.

## Plugins

|Plugin|Description|
|-|-|
|[research-note](https://github.com/eruto-skills/research-note)|Research any topic from web/YouTube/PDF/X/GitHub, then generate a footnoted Markdown note with an optional Mermaid mindmap|
|[marp](https://github.com/eruto-skills/marp)|Create, edit, and manage Marp (Markdown) presentation slides with a design-first approach|
|[nanj-thread](https://github.com/eruto-skills/nanj-thread)|Generate a Japanese anonymous-forum (なんJ) style thread for creative writing|
|[analyze-video](https://github.com/eruto-skills/analyze-video)|Analyze a YouTube reference video's structure, hooks, pacing, hedging, and rhetorical techniques|
|[review-content](https://github.com/eruto-skills/review-content)|Review video / presentation / narration content quality on 18 axes with A/B/C/D scorecard|
|[autopilot](https://github.com/eruto-skills/autopilot)|Scaffold long-running, hands-off Claude Code sessions with user-observable acceptance criteria and halt-on-uncertainty defaults|
|[design-deai](https://github.com/eruto-skills/design-deai)|Detect AI slop patterns in UI source code and generate a DESIGN.md to prevent recurring AI-generated anti-patterns|
|[design-brief](https://github.com/eruto-skills/design-brief)|Define visual design direction for a Remotion presentation — proposes color palette, typography, animation language, and generates a theme file|
|[ui-visual-reviewer](https://github.com/eruto-skills/ui-visual-reviewer)|Audit a web app's rendered UI colors using getComputedStyle — calculates WCAG contrast ratios and validates against design tokens|
